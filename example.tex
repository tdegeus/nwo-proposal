\subsection{Citations}
% ==============================================================================

A citation is included using \verb|\cite{...}|, \verb|\citep{...}|, and  \verb|\citet{...}|. The bibliography is included by \verb|\bibliography{...}|

\begin{mdframed}[backgroundcolor=black!10]
\vspace*{1em}
\begin{verbatim}
\citet{DeGeus2015b} have found that ...

\bibliography{library}
\end{verbatim}
